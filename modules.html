Project Modules

This project has declared the following modules:

Name Description
Lunifera Releng Maven - Licenses Repository Artifact A jar containing artifacts needed to resolve dependency licensing at build using license-maven-plugin.
Lunifera Releng Maven - Parent POM for Pom-First Bundles and Fragments A parent POM aimed to set up the build of bundle projects using Maven-Bundle-Plugin/Bnd.
Lunifera Releng Maven - Parent Tycho POM for Manifest-First Bundles and Fragments A parent POM aimed to set up the build of manifest-first based bundles and fragments projects using Eclipse Tycho ().
Lunifera Releng Maven - Fragment POM for Integration Tests with Equinox (Luna) container A POM aimed to group all dependencies needed for integration tests running in Equinox (Luna) container.
Lunifera Releng Maven - Fragment POM for Integration Tests with Felix container A POM aimed to group all dependencies needed for integration tests running in Felix container.
Lunifera Releng Maven - Fragment POM for Node Integration Tests with Lunifera Kernel. A POM aimed to group all dependencies needed for integration tests based on Lunifera Kernel container.
